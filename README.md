# vue-h5-template

基于vue-cli3.0+webpack 4+vant ui + sass+ rem适配方案+axios封装，构建手机端模板脚手架
- Vue-cli4    
- VantUI组件按需加载       
- Sass         
- Webpack 4        
- Vuex
- Axios封装
- rem适配方案
- 多环境配置
- 生产环境cdn优化首屏加速
- babel低版本浏览器兼容
- Eslint+Pettier统一开发规范
